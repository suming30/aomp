# AOMP-Server 项目规划

## 项目目标
构建自动化运维脚本执行管理平台后端（aomp-server），包名 com.lsm.aomp，数据库 aomp

## 阶段规划

### 阶段1：项目初始化 ✅
- 创建 SpringBoot 项目结构
- 配置 pom.xml 依赖
- 配置 application.yml

### 阶段2：数据库设计与建表
- 设计所有表结构（用户、角色、权限、主机、脚本、任务、日志、巡检、通知等）
- 直接在 MySQL 中创建数据库和表
- 插入初始数据（内置角色、权限、管理员账号）

### 阶段3：基础设施层
- 统一响应格式 Result<T>
- 全局异常处理 GlobalExceptionHandler
- i18n 国际化支持（中/英双语）
- JWT 认证与拦截器
- MyBatis-Plus 配置
- 审计日志 AOP

### 阶段4：用户与权限模块
- 用户 CRUD、登录/登出
- 角色 CRUD、权限配置
- RBAC 鉴权拦截
- 临时授权、密码策略

### 阶段5：主机资产管理模块
- 主机 CRUD、分组/标签
- SSH 连接配置
- 连通性检测
- 批量导入导出

### 阶段6：脚本仓库模块
- 脚本 CRUD、版本管理
- 危险命令检测
- 权限范围控制

### 阶段7：任务执行引擎模块
- 任务创建与执行
- 异步执行队列
- 实时日志推送
- 执行控制（暂停/终止/重试）

### 阶段8：一键巡检模块
- 巡检项配置
- 批量巡检执行
- 报告生成与导出

### 阶段9：审计日志与安全拦截
- 全操作审计日志
- 危险命令拦截
- 双人审批

### 阶段10：通知模块
- 站内信
- WebHook 通知

## 技术栈
- Java 17 + SpringBoot 3.x
- MyBatis-Plus
- MySQL 8.x
- JWT (jjwt)
- Hutool 工具库
- JSch (SSH连接)
- EasyExcel (导出)
